# Prompt

**Design a modern, responsive Admin Dashboard UI for the PrayogBharti Foundation Marketing Website.**

The admin panel should maintain the same branding, colors, typography, and visual identity as the PrayogBharti website. The interface should feel clean, professional, minimal, trustworthy, and suitable for a nonprofit organization focused on education, research, mentorship, scholarships, and community development. Use the website's green, white, light gray, and subtle accent colors to create consistency with the public-facing website. The organization emphasizes education, research & development, mentorship, inclusivity, and community empowerment, so the dashboard should reflect those values through a calm, approachable design. ([prayogbharti.org][1])

## Design Style

* Modern SaaS Admin Dashboard
* Premium UI similar to PMS Dashboard
* Large spacing
* Soft shadows
* Rounded corners (12–16px)
* Clean cards
* Professional data tables
* Glassmorphism only for small widgets
* Light Theme
* Fully responsive
* Material Icons / Lucide Icons
* Auto Layout enabled
* 8px spacing system
* Desktop Width: 1440px
